# ClusterSourMash

Usage: ./ClusterSourMash.sh <input_dir_with_fasta> [N_JOBS]

Config: specify path to sourmash executable in ./ClusterSourMash.sh and path to this directory

Dependencies: [sourmash](https://github.com/sourmash-bio/sourmash), python libs [numpy, pandas, seaborn, matplotlib, scipy]
